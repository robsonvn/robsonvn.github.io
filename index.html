O que é um Database Link

Database Link é uma conexão unidirecional que permite dois bancos de dados físicos se comunicarem como um banco lógico. (http://docs.oracle.com/cd/B28359_01/server.111/b28310/ds_concepts002.htm#ADMIN12083). Sendo assim, é possível consultar e fazer joins entre tabelas de bases distintas no mesmo ambiente, usando a mesma conexão. Inserts, updates, deletes, procedures e funções também podem ser executadas.
